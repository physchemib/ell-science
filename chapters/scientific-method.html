Linking scientific statements

Complete each sentence using because, so, or which suggests that.

Language focus
Use so when a fact comes before the consequence. Use because when the consequence comes before the fact. Use which suggests that when an observation leads to a possible explanation or hypothesis.
The temperature increased during the test, so the reaction happened faster.
